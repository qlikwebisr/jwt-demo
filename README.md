# Demo for JWT on Qlik Sense Saas
Based on Node.js

## Steps
- Creates JWT IDp and saves login details
- Generates JWT token and opens qlik sense application in iframe

## Settings
In .env file

### Installation and activation
```
npm install

node app.js
or
npm start
```
